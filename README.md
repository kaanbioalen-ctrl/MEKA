# NEON MINING GAME - Project Structure

## Klasorler
- `docs/`: GDD, roadmap, sprint ve teknik notlar
- `scenes/`: Godot sahneleri
- `scripts/`: GDScript kodlari
- `data/`: Resource tabanli gameplay verileri
- `assets/`: gorsel/ses varliklari
- `tests/`: test scriptleri
- `tools/`: yardimci scriptler
