# Yol Haritasi

## Faz 1 - Pre-Production (Hafta 1-2)
1. Vision + pillar belgesi kilitle
2. Core sistem sinirlari yaz
3. Teknik spike: zone, camera, shader grid
4. Basari metrikleri: 20 dk run, build farki, FPS stabilitesi

## Faz 2 - Playable Prototype (Ay 1-3)
1. Vertical slice: 2x2 zone, 3 asteroid, 4 enemy, 10 perk
2. Core loopu uctan uca calistir
3. Temel UI: HP/material/multiplier/perk
4. 60 FPS hedefi + death/restart akisi
5. Dis test: 10-15 dk eglenceli loop

## Faz 3 - Production (Ay 3-6)
1. Icerik genisleme: 25 zone, 8-12 enemy, 6-8 asteroid, ~30 perk
2. Enemy director + difficulty finalize
3. Perk/sinerji dengelemesi
4. VFX/SFX readability pass
5. Telemetry panel: DPS, TTK, material/min, death reason

## Faz 4 - Beta/Release (Ay 6-9)
1. Steam pipeline + store
2. QA: crash/save/softlock
3. UX polish + onboarding
4. Denge freeze
5. Release candidate + patch plani
